Comparar muestras de una Normal

Para una distribucion normal N (µ, σ2), generar una muestra aleatoria de tamaño N usando la libreria scipy.stats, a la cual llamaremos la muestra teorica. Luego, generar una segunda muestra del mismo tamaño (llamada la muestra empirica), pero usando el algoritmo de la transformada integral.
Comparar ambas muestras usando las prueba de Chi Cuadrado y la prueba de Kolmogorov-Smirnov. ¿Qu´e concluye? ¿Se
pueden considerar como muestras de una misma distribucion? Explique sus conclusiones a partir de las pruebas de hipotesis.
Use un nivel de confianza de α = 0.05
